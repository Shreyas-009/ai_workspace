## Quick Start

### 1. Clone the repository

```bash
git clone <your-github-url>
cd ai_workspace
```

### 2. Install dependencies

```bash
npm i
```

### 3. Start development server

```bash
npm run dev
```

### 4. Open in browser

Go to `http://localhost:5173` to see the application
